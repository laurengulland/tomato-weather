# tomato-weather
tiny weather exploration tool, for quickly visualizing weather forecasts in your area!

Starting with just NOAA's HRRR forecast, in a sandbox environment.
* [Weather model documentation: NOAA HRRR](http://rapidrefresh.noaa.gov/hrrr/)
* [Dataset documentation: Accessing the open-source zarr hosted by Dynamical.org](https://dynamical.org/catalog/noaa-hrrr-forecast-48-hour/)

## Installation

[Marimo setup docs](https://github.com/marimo-team/marimo?tab=readme-ov-file#quickstart) (tl;dr: `uv pip install marimo`)

## Running the exploration notebook

`marimo edit hrrr_viz_sandbox.py`

Run all cells, and optionally enter "app view" to get the streamlined interactive visualizations.
